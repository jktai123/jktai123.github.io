The purpose of this map is to demo the important components to include on a web map. This includes map details about the author, when the map was created, the map's intentions, the map's data sources, and any additional references and links.
This map also includes a legend for the data, a scale bar, zoom buttons, and support for URL query parameters that allow people to center a map based on a given URL which is handy for sharing map views.
